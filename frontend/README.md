# Frontend Application

GitHub Portfolio Generator의 프론트엔드 React 애플리케이션입니다.

## 설치

```bash
npm install
```

## 실행

### 개발 서버
```bash
npm run dev
```
http://localhost:5173 에서 실행됩니다.

### 프로덕션 빌드
```bash
npm run build
```

### 빌드 미리보기
```bash
npm run preview
```

## 주요 기능

- GitHub 사용자 검색
- 저장소 목록 표시
- AI 기반 README 요약
- 프로그래밍 언어 통계 차트
- 자동 새로고침
- 방문자 카운터

## 기술 스택

- React 18
- Vite
- Tailwind CSS
- Chart.js
- Axios
